# style9-gatsby-example

Example using [style9](https://github.com/johanholmerin/style9) with Gatsby.
See [gatsby-config.js](gatsby-config.js) for config and
[src/pages/index.tsx](src/pages/index.tsx) for usage.

## Installation

```sh
yarn
```

## Develop

```sh
yarn develop
```

## Build

```sh
yarn build
yarn serve
```
